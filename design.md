 * Non-OOP
 * Strict typed
 * Multi-file (obviously)
 * Garbage collected
 * Interpreted (for now...)
 * L and R are banned
 * Type inferrence


## Stages
 1. Tokenize
 2. Generate AST
 3. Compile
    1. Parse all files
    2. Find all structs, populate list with names and fields
    3. Resolve all argument, return, and field types
    4. Compile functions
        1. Infer all variable types
 4. Execute

## Potential UwU Rules
 * L and R are banned, always
 * /æ/ -> /hæ/
 * /s/ -> /ʃ/
 * /z/ -> /ʒ/
